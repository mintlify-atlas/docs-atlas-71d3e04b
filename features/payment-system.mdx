## Overview

SocialiCar uses PayPal as its payment processor to handle both vehicle rental payments and subscription plan purchases. The system provides a secure, user-friendly payment experience with real-time transaction processing.

## Payment Types

The platform supports two distinct payment flows:

## Payment Initialization

All payments begin through a centralized initialization endpoint:

<CodeGroup>

```php iniciar_pago.php:1-42
<?php
session_start();

if (!isset($_SESSION['usuario'])) {
    header("Location: ../../../");
    exit();
}

if (!isset($_GET['tipo'])) {
    die("Solicitud inválida");
}

$tipo = $_GET['tipo'];

switch ($tipo) {
    case 'coche':
        if (!isset($_GET['precio_coche']) || !is_numeric($_GET['precio_coche'])) {
            die("Precio del coche inválido");
        }

        $precio_coche = floatval($_GET['precio_coche']);
        $_SESSION['precio_pago'] = $precio_coche;
        $_SESSION['concepto_pago'] = "Pago de Alquiler de coche";

        header("Location: paypal.php?page=");
        exit();
    case 'basica':
        $_SESSION['precio_pago'] = 9.99;
        $_SESSION['concepto_pago'] = "Pago de Suscripción Básica";
        header("Location: paypal.php?page=planes");
        exit();
    case 'premium':
        $_SESSION['precio_pago'] = 19.99;
        $_SESSION['concepto_pago'] = "Pago de Suscripción Premium";
        header("Location: paypal.php?page=planes");
        exit();
    default:
        die("Tipo de suscripción no válido");
}
?>
```

</CodeGroup>

<Info>
Payment details are stored in the session to maintain state during the PayPal checkout flow.
</Info>

## Payment Flow Handlers

### Successful Payment

When a payment is approved:

<CodeGroup>

```javascript paypal.php:185-199
onApprove: function(data, actions){
    actions.order.capture().then(function(detalles){
        Swal.fire({
            title: '¡Pago completado!',
            text: 'Tu transacción se ha realizado con éxito.',
            icon: 'success',
            confirmButtonColor: 'var(--primary-color)'
        }).then(() => {
            if (page !== "") {
                window.location.href = "../usuario/planes.php";
            } else {
                window.location.href = "../rentacar/mostrar_coches.php";
            }
        });
    });
},
```

</CodeGroup>

<Check>Users receive a success confirmation modal before being redirected.</Check>

## Payment Flow Diagram

<Steps>
  <Step title="User Initiates Payment">
    User clicks "Alquilar" button on vehicle page or "Suscribirse" on plans page.
  </Step>
  <Step title="Payment Initialization">
    System validates user authentication, payment type, and amount.
  </Step>
  <Step title="Session Storage">
    Payment details stored in `$_SESSION` for transaction continuity.
  </Step>
  <Step title="PayPal Page">
    User redirected to payment page with PayPal button.
  </Step>
  <Step title="Order Creation">
    PayPal order created with amount and description from session.
  </Step>
  <Step title="User Authorization">
    User logs into PayPal and authorizes the payment.
  </Step>
  <Step title="Payment Capture">
    System captures the authorized payment.
  </Step>
  <Step title="Confirmation">
    Success modal displayed, user redirected to appropriate page.
  </Step>
</Steps>
