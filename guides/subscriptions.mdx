## Overview

SocialiCar offers subscription plans to help car owners maximize their rental opportunities. Premium tiers provide increased visibility, unlimited listings, and exclusive features.

<Info>
  Subscription benefits include priority placement in search results, increased ad limits, and access to premium support.
</Info>

```javascript planes.php

    // Solo activa el desplegable si el ancho es menor o igual a 700px

        card.classList.toggle('open');
    }
}
```

## Database Structure

Subscriptions are tracked in the `suscripcion_usuario` table:

```php
JOIN suscripcion_usuario sus 
    ON sus.identificacion = usuario.identificacion 
    AND sus.activo = TRUE 
    AND sus.tipo = 'Premium'
```

**Table fields:**
- `identificacion` - User ID (foreign key)
- `tipo` - Subscription type ('Premium', 'Plus', or NULL)
- `activo` - Boolean indicating active subscription
- Timestamp fields for subscription period
