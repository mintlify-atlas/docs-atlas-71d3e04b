## Overview

The Vehicles API allows users to manage their car listings on the SocialiCar platform. All endpoints require authentication.

## Create Vehicle

Create a new vehicle listing.

**Endpoint:** `POST /src/pages/coche/nuevo_coche.php`

### Authentication

Requires active session. User ID from `$_SESSION["usuario"]["identificacion"]` is used as vehicle owner.

### Basic Information

<ParamField path="matricula" type="string" required>
  Vehicle license plate. Must match Spanish format:
  - Current: 4 digits + 3 letters (e.g., `1234BCD`)
  - Old: 1-2 letters + 4-6 digits + 0-2 letters (e.g., `MA1234AB`)
</ParamField>

<ParamField path="marca" type="string" required>
  Vehicle brand. Max 50 characters. Populated from NHTSA API.
</ParamField>

<ParamField path="modelo" type="string" required>
  Vehicle model. Max 50 characters. Populated from NHTSA API.
</ParamField>

<ParamField path="anno_matriculacion" type="string" required>
  Registration year and month. Format: `YYYY-MM` (e.g., `2020-06`)
</ParamField>

<ParamField path="precio" type="float" required>
  Daily rental price in euros. Must be between 15 and 500.
</ParamField>

### Vehicle Details

<ParamField path="kilometros" type="integer" required>
  Vehicle mileage in kilometers. Must be non-negative.
</ParamField>

<ParamField path="tipo_combustible" type="string" required>
  Fuel type. Valid values: `gasolina`, `diesel`, `hibrido`, `electrico`, `glp`, `gnc`
</ParamField>

<ParamField path="tipo" type="string" required>
  Vehicle type. Valid values: `berlina`, `coupe`, `deportivo`, `furgoneta`, `monovolumen`, `suv`, `pick-up`, `roadster`, `utilitario`, `familiar`, `autocaravana`
</ParamField>

<ParamField path="transmision" type="string" required>
  Transmission type. Valid values: `manual`, `automatico`
</ParamField>

<ParamField path="color" type="string" required>
  Vehicle color. Valid values: `white`, `black`, `gray`, `red`, `blue`, `green`, `yellow`, `orange`, `brown`, `others`
</ParamField>

<ParamField path="potencia" type="integer" required>
  Engine power. Must be between 0 and 2000.
</ParamField>

<ParamField path="numero_puertas" type="integer" required>
  Number of doors. Must be between 0 and 5.
</ParamField>

<ParamField path="numero_plazas" type="integer" required>
  Number of seats. Must be between 0 and 9.
</ParamField>

<ParamField path="descripcion" type="string">
  Vehicle description. Max 200 characters.
</ParamField>

### Location

<ParamField path="direccion" type="string" required>
  Full address. Format: `Street, City, Province, Postal Code, Country`
  
  Must be in Spain with valid 5-digit postal code.
</ParamField>

<ParamField path="tipo_aparcamiento" type="string" required>
  Parking type. Valid values: `calle`, `garaje`, `parking`
</ParamField>

<ParamField path="lat" type="float">
  Latitude coordinate for vehicle location.
</ParamField>

<ParamField path="lon" type="float">
  Longitude coordinate for vehicle location.
</ParamField>

## Update Vehicle

Update an existing vehicle listing.

**Endpoint:** `POST /src/pages/coche/editar_coche.php?matricula={matricula}`

### Query Parameters

<ParamField query="matricula" type="string" required>
  License plate of vehicle to update
</ParamField>

### Get Current Vehicle Data

```sql
SELECT * FROM coche WHERE matricula = ?
```

### Get Current Extras

```sql
SELECT * FROM extras_coche WHERE matricula = ?
```

### Request Parameters

Same as Create Vehicle endpoint. Only changed fields need to be included.

### Update Queries

```sql
-- Update vehicle
UPDATE coche SET
    marca = ?, modelo = ?, anno_matriculacion = ?, kilometros = ?,
    combustible = ?, transmision = ?, direccion = ?, ciudad = ?,
    provincia = ?, codigo_postal = ?, lat = ?, lon = ?,
    tipo_aparcamiento = ?, tipo = ?, precio = ?, descripcion = ?,
    color = ?, plazas = ?, puertas = ?, potencia = ?
WHERE matricula = ?

-- Update extras
UPDATE extras_coche SET
    aire_acondicionado = ?, gps = ?, wifi = ?, sensores_aparcamiento = ?,
    camara_trasera = ?, control_de_crucero = ?, asientos_calefactables = ?,
    bola_remolque = ?, fijacion_isofix = ?, apple_carplay = ?,
    android_carplay = ?, baca = ?, portabicicletas = ?, portaequipajes = ?,
    portaesquis = ?, bluetooth = ?, cuatro_x_cuatro = ?, mascota = ?,
    fumar = ?, movilidad_reducida = ?
WHERE matricula = ?
```

### Note

License plate (`matricula`) cannot be changed during update.

## Delete Vehicle

Delete a vehicle listing and all associated data.

**Endpoint:** `GET /src/pages/coche/eliminar_coche.php?matricula={matricula}`

### Query Parameters

<ParamField query="matricula" type="string" required>
  License plate of vehicle to delete
</ParamField>

### Deletion Process

```php
// 1. Get image path
SELECT ruta_img_coche FROM imagen_coche WHERE id_coche = ? LIMIT 1

// 2. Delete directory recursively
function eliminarDirectorio($ruta) {
    $archivos = array_diff(scandir($ruta), ['.', '..']);
    foreach ($archivos as $archivo) {
        $ruta_completa = $ruta . '/' . $archivo;
        if (is_dir($ruta_completa)) {
            eliminarDirectorio($ruta_completa);
        } else {
            unlink($ruta_completa);
        }
    }
    rmdir($ruta);
}

// 3. Delete from database (CASCADE handles related tables)
DELETE FROM coche WHERE matricula = ?
```

### Response

Redirects to user profile page on success.

## Get Vehicle Models

Get available models for a specific vehicle brand.

**Endpoint:** `GET /src/pages/coche/obtener_modelos.php?marca={marca}`

### Query Parameters

<ParamField query="marca" type="string" required>
  Vehicle brand name
</ParamField>

### External API Integration

Uses NHTSA (National Highway Traffic Safety Administration) API:

```php
// Get all makes
$apiUrlMarcas = "https://vpic.nhtsa.dot.gov/api/vehicles/GetMakesForVehicleType/car?format=json";

// Get models for specific make
$apiUrlModelos = "https://vpic.nhtsa.dot.gov/api/vehicles/GetModelsForMakeId/{$makeId}?format=json";
```

### Response

```json
[
    "Corolla",
    "Camry",
    "RAV4",
    "Prius",
    "Highlander"
]
```

## Validation Rules

### License Plate Validation

```php
// Current format: 1234BCD
$patron_actual = "/^[0-9]{4}[BCDFGHJKLMNPRSTVWXYZ]{3}$/";

// Old format: MA1234AB
$patron_antiguo = "/^[A-Z]{1,2}[0-9]{4,6}[A-Z]{0,2}$/";

if (!preg_match($patron_actual, $matricula) && !preg_match($patron_antiguo, $matricula)) {
    $err_matricula = "La matrícula no es válida";
}
```

### Address Validation

```php
// Split address by commas
$partes_direccion = explode(",", $direccion);

// Must have at least 5 parts
if ($total < 5) {
    $err_direccion = "La dirección debe tener al menos 5 partes";
}

// Validate postal code (5 digits)
if (!is_numeric($cp) || strlen($cp) != 5) {
    $err_direccion = "El código postal no tiene un formato válido";
}

// Must be in Spain
if (strtolower($pais) !== "españa") {
    $err_direccion = "La dirección debe estar en España";
}
```
