## For Vehicle Owners

<Accordion title="List Your Vehicle">
  - Upload multiple photos of your car
  - Set your own pricing (€15-€500/day)
  - Specify 20+ vehicle features (GPS, Bluetooth, A/C, etc.)
  - Define your availability schedule
  - Choose your parking location type
  - Full control over booking requests
</Accordion>

<Accordion title="Manage Bookings">
  - View incoming rental requests
  - Accept or decline bookings
  - Communicate with renters via chat
  - Set minimum and maximum rental periods
  - Update availability in real-time
</Accordion>

<Accordion title="Earn Income">
  - Keep majority of rental fees
  - Set custom pricing based on demand
  - Offer weekly or monthly discounts
  - Track earnings through dashboard
  - Receive payments via PayPal
</Accordion>

<Accordion title="Premium Features">
  - Priority placement in search results
  - Gold border highlighting on listings
  - Premium badge visibility
  - Extended photo galleries
  - Priority customer support
</Accordion>

## For Renters

<Accordion title="Find the Perfect Car">
  - Search by city and province
  - Filter by dates and availability
  - Browse by vehicle type (SUV, Sedan, Van, etc.)
  - Filter by fuel type (Electric, Hybrid, Gas, Diesel)
  - Set price range preferences
  - Check 20+ equipment features
</Accordion>

<Accordion title="Book with Confidence">
  - View detailed vehicle specifications
  - Read reviews from previous renters
  - See high-quality photos
  - Check real-time availability
  - Communicate directly with owner
  - Secure payment processing
</Accordion>

<Accordion title="During Your Rental">
  - Access owner contact information
  - Chat support for questions
  - Vehicle location and pickup details
  - Insurance information
  - 24/7 platform support
</Accordion>

<Accordion title="After Your Trip">
  - Leave reviews and ratings
  - Report any issues
  - Save favorite vehicles
  - Rebook easily for next trip
</Accordion>

## Integration Features

## Responsive Design

SocialiCar is fully responsive and optimized for all devices:

- **Desktop**: Full-featured interface with multi-column layouts
- **Tablet**: Optimized views with collapsible navigation
- **Mobile**: Touch-friendly interface with bottom navigation
- **All Devices**: Bootstrap 5 responsive grid system
